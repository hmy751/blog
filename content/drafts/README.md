# Drafts

이 폴더의 글은 공개 전 초안이다. `content/posts/`로 옮기기 전까지는 발행 후보로만 다룬다.

## 승격 기준

- `date: TBD`를 실제 발행일로 바꾼다.
- frontmatter와 파일명은 [prepublish-check.md](../../editorial/guards/prepublish-check.md)의 Draft -> Post 승격 기준을 따른다.
- `node scripts/blog-prepublish-check.mjs`를 통과한다.
- `blog-tone-critic`, `blog-structure-critic`, `blog-evidence-checker` 검토에서 막는 이슈가 없다.
- 최종 발행 여부를 사용자가 결정한다.
